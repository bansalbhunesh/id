# Submission Deck

`index.html` is the full 13-slide UdyamPulse pitch deck, mapped 1:1 to the official IDBI Innovate template's required sections (Team Details, Brief, Opportunities, Features, Process Flow, Wireframes, Architecture, Technologies, Cost, Snapshots, Performance Report, Additional Details, Links).

Self-contained — fonts and styling are inlined, no external dependencies or internet connection needed to view it.

## To get the mandatory PDF

1. Open `index.html` in Chrome (double-click, or serve it locally and navigate to it).
2. Before printing, replace the placeholder stock-photo slides (6 and 10) and the "add link" placeholders in slide 13 with your real screenshots and links.
3. Print (`Ctrl+P`) → Destination: **Save as PDF** → the print stylesheet renders each slide at a fixed 1280×720 (16:9), one per page.
4. Upload the resulting PDF to the H2S submission form.

## Notes

- This deck was built independently (not by editing IDBI's official template file directly — no PPTX/InDesign editing tool was available), but follows its exact section structure and order so content maps straight across if you'd rather paste text into the official template instead.
- `docs/PITCH_OUTLINE.md` has the same content in plain text if you need to copy-paste into the official template file instead of using this HTML version.
